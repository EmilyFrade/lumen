# **CSI606-2025-01 - Proposta de Trabalho Final**

## *Discente:* Emily Frade dos Santos - 23.1.8001

### Resumo
O presente trabalho consiste no desenvolvimento do **Lumen**, um sistema web destinado ao gerenciamento pessoal de cursos. A aplicação tem como objetivo oferecer aos usuários um ambiente simples e intuitivo para cadastro, consulta, organização e avaliação de cursos.

---

### 1. Tema
O trabalho final tem como tema o desenvolvimento do **Lumen – Sistema Web para Gerenciamento de Cursos**.

---

### 2. Escopo
Este projeto terá as seguintes funcionalidades:
- Autenticação de usuários (cadastro e login).
- CRUD de cursos (criação, visualização, edição e exclusão).
- Organização dos cursos por categorias.
- Acompanhamento do progresso, adição de anotações e avaliação simples de cursos.

---

### 3. Restrições
Neste trabalho não serão considerados:
- Algoritmos de recomendação de cursos.
- Integração com plataformas externas (ex.: Udemy, Coursera).
- Mecanismos de notificações ou envio de e-mails.
  
Essas funcionalidades serão consideradas em etapas posteriores do projeto (TCC).

---
